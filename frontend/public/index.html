Browser Extensions
Add jobs to Pursuit in one click from any job site — LinkedIn, Indeed, Greenhouse, Lever, Workday, and more.
Download Extension (.zip)
Contains files for both Chrome and Safari. Auto-detects job info from LinkedIn, Indeed, Greenhouse, Lever, and Workday.
Chrome
  1. Download and unzip
  2. Go to chrome://extensions
  3. Enable Developer Mode
  4. Click Load unpacked
  5. Select the applied-extension folder
  6. Sign in with your Pursuit credentials
Safari
Requires Xcode
  1. Install Xcode (free)
  2. Download and unzip
  3. Double-click Install Pursuit Extension.app
  4. Click Install — fully automated
  5. Enable Pursuit in Safari → Extensions
Keep the companion app in your Dock while using the extension.
Help & Support
USER GUIDE
Adding a job
Click + Add application → paste a job URL and click Open to auto-parse title, company, and location. Fill in Work Type and click Save.

Tailoring tab
1. Select Resume or Cover Letter → 2. Upload or pick from Library → 3. Add optional context → click Tailor → 4. Edit result → Download .docx or .txt.

Documents tab (per job)
All documents for each application. Upload files, add from your Library, and mark one as Used for application. Syncs to Document Library.

Document Library
Sidebar button. Search and filter all resumes and cover letters. Upload, create blank docs, or edit in the built-in editor.

Insights tab
Click Research this role for AI-powered company research — culture, red/green flags, LinkedIn strategy, news, interview tips. Add a Finnhub key for live stock data.

Analytics & Check Postings
Open Analytics from the sidebar. Click ↺ Check postings in the Analytics header to scan all active job URLs for expired signals. Expired jobs are tagged orange.

Browser Extensions
Install from Settings → Browser Extensions to add jobs directly from any job site in one click.
Data & Export
EXPORT ALL DATA
Download all your job applications as a zip. Each job gets its own folder with metadata, notes, resume, cover letter, and tailored documents.
