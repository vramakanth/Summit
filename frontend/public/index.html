BROWSER EXTENSION
Add jobs to Summit in one click from any job site — LinkedIn, Indeed, Greenhouse, Lever, Workday, and more.
Works on Chrome, Brave, Edge, Arc, and any Chromium browser. Auto-detects job info from LinkedIn, Indeed, Greenhouse, Lever, Workday, and more.
Chrome
  1. Download and unzip
  2. Go to chrome://extensions
  3. Enable Developer Mode
  4. Click Load unpacked
  5. Select the summit-extension folder
  6. Sign in with your Summit credentials
Safari
Requires Xcode
  1. Install Xcode (free)
  2. Download and unzip
  3. Double-click Install Summit Extension.app
  4. Click Install — fully automated
  5. Enable Summit in Safari → Extensions
Keep the companion app in your Dock while using the extension.
